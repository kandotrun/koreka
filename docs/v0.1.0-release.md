# v0.1.0 Release Notes

## Koreka（これか！）— v0.1.0

リリース日: 2026-03-21

---

### 新機能

#### ゲームプレイ
- **カテゴリ選択**: ルーム作成時に9種類のカテゴリから好みのジャンルを選択可能
- **カスタムデッキ**: 自分で作ったお題（最大50個）でプレイ可能
- **AI オンデマンド生成**: Gemini API を使ったお題のリアルタイム生成
- **選択制限時間（30秒）**: 選択フェーズに30秒のタイマーを追加。タイムアウト時はランダム自動選択
- **クイックリプレイ**: 結果画面からホストが同じルームで「もう一回」プレイ可能
- **プレイヤーキック**: ホストがロビーで不要なプレイヤーを退出させる機能
- **思い出記録**: 結果のお題にコメントを残して保存

#### UI/UX
- **OGP メタタグ**: SNS シェア時にプレビュー画像とテキストを表示
- **PWA アイコン**: ホーム画面追加時の高品質アイコン
- **シェアボタン**: 結果をネイティブシェア API またはクリップボードで共有
- **エラー UI**: 満員・ゲーム中の参加エラーを分かりやすく表示
- **Error Boundary**: 予期しないエラーをキャッチしてフレンドリーな画面を表示
- **アクセシビリティ**: aria-label、aria-live、role="main" などの ARIA 属性を全画面に追加
- **サウンド UI**: スワイプ・ラウンド完了・結果発表のサウンドエフェクト（Web Audio API）
- **favicon**: ブラウザタブ用アイコン

#### インフラ・技術
- **多言語対応 (i18n)**: 日本語・英語・韓国語の3言語サポート
- **管理者認証**: パスワードベースの管理画面ログイン
- **管理者統計**: お題カード数・ルーム数・思い出数・人気お題 TOP 20
- **ルーム自動削除 (TTL)**: 2時間後にルームを自動クリーンアップ
- **SWR リファクタリング**: Home・Admin の API フェッチを useSWR に移行
