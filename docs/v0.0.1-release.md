# Koreka v0.0.1 リリースノート

**リリース日**: 2026-03-21
**タグ**: [v0.0.1](https://github.com/kandotrun/koreka/releases/tag/v0.0.1)

## 概要

Koreka MVP の初回リリース。リアルタイムマルチプレイヤーカードゲームの基本フローが一通り動作する状態。

## 実装済み機能

### 画面一覧

#### 1. Home（トップ画面）
```
┌──────────────────────────┐
│         Koreka            │
│  これか！と思える瞬間を。  │
│                          │
│   [ルームを作る 🎴]       │
│   [コードで参加]          │
│                          │
│  → 名前入力              │
│  → 4桁コード入力 (OTP式)  │
└──────────────────────────┘
```
- ルーム作成: 名前入力 → POST /api/rooms → ルーム画面に遷移
- コード参加: 名前 + 4桁コード → ルーム画面に遷移
- framer-motion アニメーション

#### 2. Lobby（待機室）
```
┌──────────────────────────┐
│  ルーム                   │
│     7 2 8 4              │
│  このコードを友達にシェア  │
│  ──────────────────────  │
│   🟢 カン (ホスト)        │
│   🟢 タロウ               │
│   ⚪ ハナコ               │
│                          │
│   [ゲーム開始 ▶]          │
└──────────────────────────┘
```
- WebSocket接続 + 自動join
- 参加者リスト（リアルタイム更新）
- イニシャルアバター（カテゴリカラー）
- 準備OK / 待機中 ステータス
- ホストのみゲーム開始ボタン表示

#### 3. Game（選別フェーズ）
```
┌──────────────────────────┐
│  Round 2    残り 3/5      │
│  ● ● ○ ○                │
│                          │
│   ┌────────────────────┐ │
│   │   🏔️ 冒険          │ │
│   │   タクシーで        │ │
│   │   行けるとこまで    │ │
│   │   行く              │ │
│   └────────────────────┘ │
│  ← やらない    やる →     │
└──────────────────────────┘
```
- Tinder風スワイプUI（タッチ + マウス対応）
- Spring physics アニメーション
- 方向インジケーター（「やる!」/「パス」）
- プログレスバー
- 他プレイヤー進捗ドット
- カード回し（右回転）
- 収束判定 → 最終投票へ

#### 4. 最終投票
```
┌──────────────────────────┐
```

- 残ったカード一覧表示
- タップで投票
- 二重投票防止
