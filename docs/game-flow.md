# Game Flow

## 1. ルーム作成

```
ホスト: [ルーム作成] ボタンタップ
  → Workers: POST /api/rooms
  → Durable Object 生成
  → 4桁コード発行 (例: "7284")
  → ホストに QRコード + コード表示
```

## 2. 参加

```
ゲスト: コード入力 or QR読み取り
  → /{code} に遷移
  → WebSocket接続
  → 名前入力 → { type: 'join', name: 'タロウ' }
  → 全員に { type: 'players', players: [...] }
```

## 3. 待機室 (Lobby)

```
参加者一覧が表示される
各自 [準備OK] をタップ → { type: 'ready' }
全員 ready → ホストに [ゲーム開始] ボタン表示
```

## 4. カード配布 (Deal)

```
ホスト: [ゲーム開始] タップ
  → Durable Object:
    1. カードプールから N枚ランダム抽出
       N = playerCount * cardsPerPlayer (default: 5)
    2. シャッフル
    3. 各プレイヤーに均等配布
  → 各プレイヤーに { type: 'deal', cards: [...] }
```

## 5. 選別フェーズ (Select)

```
各プレイヤーの画面:
  ┌─────────────────────┐
  │                     │
  │   🎴 カード表示      │
  │   「夜の海に行く」    │
  │                     │
  │  ← やらない  やる →  │
  │  (swipe left/right) │
  │                     │
  │   残り: 3/5枚       │
  └─────────────────────┘

ルール:
- 全カードをスワイプで仕分け
- 「やる」は何枚でもOK（ただし最低1枚は残す）
- 「やらない」は捨てられる
- 全カード仕分けたら自動送信
  → { type: 'select', cardIds: ['card-1', 'card-3'] }
```

## 6. カード回し (Pass)

```
全員の選択完了
  → Durable Object:
    1. 各プレイヤーの「残した」カードを収集
    2. 右隣のプレイヤーに渡す
       Player A の残り → Player B
       Player B の残り → Player C
       ...
    3. round++
  → 各プレイヤーに { type: 'pass', cards: [...], round: N }

次の選別フェーズへ（5に戻る）
```

## 7. 収束判定

```
毎ラウンド終了時に判定:
  残りカード総数 <= playerCount の場合 → 最終投票へ
  それ以外 → 次のラウンドへ
```

## 8. 最終投票

```
各プレイヤーに残り全カードが表示される
各自 1枚だけ選ぶ（タップで選択）

全員選択完了
  → 最多得票のカードが結果
  → 同率の場合はランダム
```

## 9. 結果発表 🎉

```
全員の画面に同時表示:

  ┌─────────────────────────┐
  │                         │
  │      これか！            │
  │                         │
  │   ┌─────────────────┐   │
  │   │                 │   │
  │   │  「タクシーで    │   │
  │   │   行けるところ   │   │
  │   │   まで行く」     │   │
  │   │                 │   │
  │   └─────────────────┘   │
  │                         │
  │   3/4人が選択 🔥         │
  │                         │
  │  [もう一回]  [思い出記録] │
  └─────────────────────────┘
```

## タイムライン（想定）

| Phase | 所要時間 |
|-------|---------|
| ルーム作成〜参加 | 1-2分 |
| 1ラウンドの選別 | 30秒-1分 |
| ラウンド数（4人の場合） | 3-5回 |
| 最終投票 | 30秒 |
| **合計** | **5-10分** |
